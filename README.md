# HRFlow EMS

A modern Human Resources Management System built with ASP.NET Core.

---

## 🚀 Project Goals

- Employee Management
- Department Management
- Leave Management
- Role Based Authorization
- Audit Logging
- Dashboard & Reports
- Excel Import / Export
- Real-time Notifications
- REST API
- Clean Architecture

---

## 🛠️ Technologies

- ASP.NET Core
- Entity Framework Core
- SQL Server
- Bootstrap 5
- jQuery
- AutoMapper
- FluentValidation
- Serilog
- SignalR
- Hangfire
